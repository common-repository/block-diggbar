This plugin blocks your blog from being accessed by the DiggBar.

== Description ==

This plugin blocks your blog from being accessed by the DiggBar. This message is displayed to the user:

The DiggBar is blocked on this blog. [Read why](http://daringfireball.net/2009/04/how_to_block_the_diggbar).
